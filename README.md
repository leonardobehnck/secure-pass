# SecurePass

A simple React Native (Expo) mobile app that generates random secure passwords on demand. Built as a learning project, it features a clean UI where users can tap "Gerar" to generate a new password and "Copiar" to copy it to the clipboard, with the result displayed in a read-only input field. The app uses Expo's clipboard API, safe area context for proper layout across devices, and follows a component-based architecture with separate folders for screens (Home), reusable components (Button, InputText, Logo, Footer, Menu), and services (password generation logic).
